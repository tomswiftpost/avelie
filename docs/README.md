# Avelie GitHub Pages Site

This folder is a static GitHub Pages site for App Store support, marketing, privacy, and terms URLs.

## GitHub Pages Setup

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/docs` folder.
6. Save.

If the repository is named `avelie` under the `tomswiftpost` account, the URLs should be:

- Marketing URL: `https://tomswiftpost.github.io/avelie/`
- Support URL: `https://tomswiftpost.github.io/avelie/support.html`
- Privacy Policy URL: `https://tomswiftpost.github.io/avelie/privacy.html`
- Terms URL: `https://tomswiftpost.github.io/avelie/terms.html`

If the repository has a different name, replace `avelie` in the URL with the repository name.
